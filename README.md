## ✏️ 컨벤션

### Commit Message Convention

|    Type     | Description                                       |
| :---------: | ------------------------------------------------- |
|   `Feat:`   | 새로운 기능 추가                                  |
|   `Fix:`    | 버그 수정                                         |
|   `Docs:`   | 문서 수정                                         |
|  `Style:`   | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 |
| `Refactor:` | 코드 리팩토링                                     |
|   `Test:`   | 테스트 코드 및 리펙토링 테스트 코드 추가          |
|  `Design:`  | 사용자 UI 디자인 변경 (CSS)                       |
